== Description ==

Add jQuery-driven fancy slideboxes to your WordPress pages and Posts. With this plugin a layer will slide on an inserted picture on mouse hover. You can find detailed documentation about the plugin on the author's homepage:
[http://arachnoidea.com/wp-slide-up-box/](http://arachnoidea.com/wp-slide-up-box/)

== Installation ==

1. **Download** the zip package from the plugin’s download page.
1. Go to your **WordPress Admin area** and sign in (e.g.: www.yourdomain.com/wp-admin)
1. Go to **Plugins » Add new**, then click on Upload
1. Select the zip package you downloaded in (1) then click on **Install Now** button.
1. If the upload process was successful, you can now **Activate the plugin**.
1. After you installed and activated the plugin, a new submenu item will appear in the WP-admin area (You’ll be able to find it after installation in **Appearance » WP Slide-up box**)
